PAWS

PAWS는 AI와 데이터를 기반으로 반려인의 일상, 산책 매칭, 커뮤니티, 실시간 채팅, 반려견 거래 서비스를 하나로 연결한 스마트 반려견 라이프 통합 플랫폼입니다.

PROJECT OVERVIEW

반려인의 모든 경험을
하나로 연결하다

기존 반려 서비스는 커뮤니티, 거래, 산책, 추천 기능이 각각 분리되어 있었습니다. PAWS는 이를 AI와 데이터 기반으로 통합하여 하나의 연결된 반려 라이프 플랫폼을 제공합니다.

PROBLEM DEFINITION

기존 반려 서비스의
한계를 해결하다

😢

신뢰 부족

사용자 활동 데이터와 리뷰가 연결되지 않아 안전한 산책 및 거래 연결이 어려웠습니다.

📚

정보 분산

커뮤니티, 마켓, 견종 정보, 산책 서비스가 각각 분리되어 있었습니다.

🤖

AI 추천 부재

사용자 성향과 활동 데이터를 활용한 개인화 추천 구조가 부족했습니다.

⚡

실시간 연결 부족

사용자 간 즉각적인 연결과 실시간 소통 기능이 부족했습니다.

MAIN FEATURES

PAWS 주요 기능

AI와 데이터를 기반으로 반려인의 일상 경험을 하나의 플랫폼으로 연결합니다.

📍

산책 매칭 시스템

사용자 위치 기반으로 산책 모집 및 신청, 수락/거절 기능을 제공합니다. 매너온도를 활용해 신뢰도 높은 산책 연결을 지원합니다.

🤖

AI 추천 시스템

OpenAI GPT 기반으로 사용자 활동, DBTI, 채팅 데이터를 분석하여 개인화 추천 서비스를 제공합니다.

🧠

DBTI 성향 분석

반려견의 활동성, 사회성, 성향 데이터를 기반으로 사용자 간 궁합 분석과 매칭 품질을 향상시킵니다.

💬

실시간 채팅

매칭 수락 시 자동 채팅방이 생성되며 사용자 간 실시간 메신저 기능을 제공합니다.

🛒

PAWS 마켓

반려견 상품 거래, 장바구니, 주문, 리뷰 기능을 제공하며 리뷰 기반 신뢰 시스템을 지원합니다.

📝

커뮤니티 시스템

자유 게시판, 질문 게시판, 정보 공유 게시판을 통해 사용자 간 소통과 정보 공유를 지원합니다.
