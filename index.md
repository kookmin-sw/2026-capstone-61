# 📌 프로젝트 소개

PAWS는 반려견 보호자를 위한:

- 📍 위치 기반 산책 매칭
- 🤖 AI 기반 추천 시스템
- 🧠 DBTI 성향 검사
- 💬 실시간 메신저
- 🛒 반려견 마켓
- 🐶 견종 사전
- 📝 커뮤니티

기능을 하나의 플랫폼에서 제공하는 통합 반려견 서비스입니다.
